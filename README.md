# Современный осциллограф - 4 канала

Веб-приложение осциллографа, созданное с использованием Flask и HTML5 Canvas для визуализации и анализа сигналов в реальном времени.

## Возможности

- **4 независимых канала**: Одновременный просмотр и управление до 4 каналами сигнала
- **Множество типов волн**: Генерация синусоидальных, квадратных, треугольных, пилообразных сигналов, шума и композитных сигналов
- **Визуализация в реальном времени**: Отображение сигналов в прямом эфире с помощью HTML5 Canvas
- **Режим спектрограммы**: Анализ в частотной области с интерактивной спектрограммой
- **Управление сигналом**: Настройка частоты, амплитуды, смещения и уровня шума для каждого канала
- **Экспорт данных**: Сохранение захваченных данных в формате CSV
- **Поддержка загрузки**: Загрузка ранее сохранённых конфигураций сигналов
- **Пауза/возобновление**: Управление генерацией сигналов с функцией паузы
- **Современный интерфейс**: Тёмная тема с адаптивным дизайном

- Python 3.8+
- Flask
- NumPy
- SciPy
- Matplotlib

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone <url-репозитория>
   cd <каталог-проекта>
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Использование

1. Запустите сервер Flask:
   ```bash
   python app.py
   ```

2. Откройте браузер и перейдите по адресу:
   - Основной осциллограф: `http://localhost:5000`
   - Просмотр спектрограммы: `http://localhost:5000/spectrogram`

3. Используйте элементы управления для:
   - Включения/выключения отдельных каналов
   - Выбора типа волны для каждого канала
   - Настройки частоты, амплитуды, смещения и уровня шума
   - Паузы/возобновления генерации сигнала
   - Экспорта данных в CSV

## API эндпоинты

| Эндпоинт | Метод | Описание |
|----------|--------|-------------|
| `/` | GET | Основной интерфейс осциллографа |
| `/spectrogram` | GET | Визуализация спектрограммы |
| `/data` | GET | Получение текущих данных сигнала (JSON) |
| `/config` | GET | Получение конфигурации каналов |
| `/config` | POST | Обновление конфигурации каналов |
| `/export` | GET | Экспорт данных сигнала в формате CSV |
| `/upload` | POST | Загрузка конфигурации сигнала |

## Конфигурация

Каждый канал поддерживает следующие параметры:

- **type**: Тип волны (`sine`, `square`, `triangle`, `sawtooth`, `noise`, `composite`)
- **frequency**: Частота сигнала в Гц
- **amplitude**: Амплитуда сигнала
- **offset**: Постоянное смещение (DC offset)
- **noise**: Уровень шума (0-1)
- **enabled**: Состояние включения/выключения канала

## Настройки каналов по умолчанию

| Канал | Тип | Частота | Амплитуда | Смещение |
|-------|------|-----------|-----------|--------|
| CH1 | Синусоида | 10 Гц | 1.0 | 0.0 |
| CH2 | Квадрат | 2 Гц | 3.0 | 2.0 |
| CH3 | Треугольник | 2.5 Гц | 4.0 | -2.0 |
| CH4 | Пилообразный | 3 Гц | 2.5 | 1.0 |
